  # Raworc
  
  **Remote Agentic Work Orchestrator**

## 🚀 Get Started

**End Users (Production):**
```bash
npm install -g @raworc/cli
raworc start
raworc login -u admin -p admin
raworc session
```

**Contributors (Development):**  
```bash
git clone <this-repo>
cd raworc
./scripts/link.sh
./scripts/build.sh
./scripts/start.sh
```
