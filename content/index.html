How to Use the Content Folder

When you need to display information to users visually, always use the /agent/content/ folder:

// Example: Creating a data visualization page text_editor("create", "content/data-report.html", ` <!DOCTYPE html> <html> <head> <title>Data Analysis Report</title> <script src="https://cdn.jsdelivr.net/npm/chart.js"></script> </head> <body> <h1>Sales Analysis Report</h1> <canvas id="salesChart"></canvas> <script> // Chart implementation here </script> </body> </html> `) // Always update index.html to link to new content text_editor("str_replace", "content/index.html", '<a href="data-report.html">View Sales Report</a>')
